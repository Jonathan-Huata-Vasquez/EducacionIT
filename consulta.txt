COMANDOS

GIT INIT (inicializa un repositorio de GIT)

GIT INIT "Nombre de Carpeta" (Crea un Repositorio y lo inicializa)

GIT CONFIG --GLOBAL USER.NAME "Usuario" (Define el Nombre de Usuario para todos los proyectos)

GIT CONFIG --GLOBAL USER.EMAIL "correo" (Define el Correo de Usuario para todas las configuraciones de Repositorio)

GIT ADD . (Agrega todos los archivos y carpetas del Directorio Actual al STAGING AREA
)

GIT STATUS (MUESTRA LOS ARCHIVOS Y CARPETAS QUE SE ENCUENTRAN EN EL DIRECTORIO, ASI COMO TAMBIEN EL ESTADO Y SEGUIMIENTO DE LOS MISMOS)
